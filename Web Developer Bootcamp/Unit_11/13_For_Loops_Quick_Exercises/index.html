For Loops Quick Exercises

Example
for(let i = 0; i < 16; i+=8) {
console.log(i)
}

init: let i = 0;
condition: i < 16;
step: i+=8;

Result:
Console logs: 0, 8

Example2
const str = "ahceclvlxo"
for(let i = 1; i < str.length; i+=2) {
console.log(str[i])
}

init: let i = 1;
condition: i < str.length;
step: i+=2;

Result:
Console logs: hello
