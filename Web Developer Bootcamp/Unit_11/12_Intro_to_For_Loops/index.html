Intro to For Loops

This is what a For Loop looks like:

for(init; condition; step) {
//run some code
}

Example
for(var count = 1; count < 6; count++) {
console.log(count)
}

init: var count = 1;
condition: count < 6;
step: count++;

Example 2
const str = "hello";
for(let i = 0; i < str.length; i++) {
console.log(str[i])
}

init: let i = 0;
condition: i < str.length;
step: i++;
